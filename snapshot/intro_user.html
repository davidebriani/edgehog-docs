    Introduction

    Edgehog is an Open Source IoT platform focused on device fleet management. It conveniently handles fundamental operations such as performing update campaigns, getting device status information, geolocation and much more. It is based on Astarte.

    This documentation provides information on daily operations for Edgehog users and integrators. It goes through fundamental operations such as performing updates, querying APIs, integrating 3rd party applications, etc.
